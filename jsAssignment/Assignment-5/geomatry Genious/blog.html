Question 1: What is DOM and what is the purpose of DOM?

Answer 1...

The meaning of DOM = Document Object Model

I can obviously say that DOM stands in the middle of Html and Java Script. DOM helps Java Script to manupulate every element, style and structure form html as an object, and also converts javaScript objects in a reverse way.

Dom rerpensents the html document as a node or as an object

Question 2: How will you select HTML elements using DOM? Name the DOM methods.

Answer 2...

By using DOM I can sellect HTML elements in four way:
  • document. getElementById ('elementId')
  • document. getElementsByClassName ('elements-class-name')
  • document.getElementsByTagName ('elements-tag-name')
  • document. querySelector and querySelectorAll ('elements-class/id')

note:querySelectorAll sellects element from the nodeList

Question 4: What is a callback function and why will you use it?

Answer 4...

A function which accepts another function as an argument

In JavaScript, functions are objects. we Can pass objects to functions as parameters also.

I use callback function to reduce repetative code, to make my code more cleaner and also to pass a function as an object or parameter to another function.
